===============
    PYTRIS
===============
Tetris for Python


Author:
	Jinyong Lee
	(gullpong9@gmail.com)

Usage:
	Run file "game.py" with Python

	$ pip install -r requirements.txt
	$ python game.py

Controls:

	Left/Right arrow: move block
	Up arrow: spin block
	Down arrow: drop block
	Esc: quit game
